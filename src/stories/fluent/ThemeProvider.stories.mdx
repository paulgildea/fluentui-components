# ThemeProvider
Theming Component that provides theming context for Fluent UI React components.

## Example usage
Use the theme with Fluent UI by wrapping content within the provider. If `theme` is not provided, default (Fluent) theme will be provided:

```tsx
import { ThemeProvider } from '@fluentui/react-theme-provider';
import { Button } from '@fluentui/react-button';

export const App = () => (
  <ThemeProvider>
    <Button>Button</Button>
  </ThemeProvider>
);
```

## Theming Concepts
Fluent UI React provides various ways to apply theming to the set of components within the library. The following outlines levels of theming:

- Global
- Component
- Instance

**It is recommended to apply theming at the higher levels as possible because it provides the best performance when applying theming across your applciation.**

**Good Recommendation**: Define color values at the Global level that apply to all the components within the library. Typical examples are brand color.

## Theme Concpet Matrix
| Theme Concept    | Global            | Component              | Instance       |
| ---------------- | ----------------- | ---------------------- | -------------- |
| Color            | palette           | semanticColors, styles, variants | styles, tokens |
| Fonts            | defaultFont, font | styles, variants       | styles, tokens |
| Elevation        | effects           | styles, variants       | styles, tokens |
| Corner Roundness | effects           | styles, variants       | styles, tokens |
| Spacing          | spacing           | styles, variants       | styles, tokens |
